Trivia Game


Skills Used: HTML5, CSS3, Javascript, Jquery

This is a small game built by myself. This game has a horror theme (the best theme possible). Upon starting the game, a question will be displayed. You will have 10 seconds to answer this question, if no answer is chosen, it will be considered a timeout, which will be displayed at the results screen at the end of the game, along with correct and incorrect answers. There is instant feedback on each question From the end game screen, you can restart and play the game again for a better score.
