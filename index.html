GameRatr


Skills Used: HTML5, CSS3, Javascript, jQuery, MySQL, Sequelize, Node.js, Express.js

GameRatr is an app designed for you to browse through a database of games. You can filter through the results to whatever you'd like. If the game that you are looking for is not found, you can add a game found on the left nav. Once a game is added, it will be inserted into the ever growing database and the page will refresh with all results, including your new game! This project was done by a group in which I was the project manager and wrote most of the code.

NewsScrapr
